## urlscan pro visibility update

Updated visibility of the scan result owned by you or your team

```
urlscan pro visibility update [flags]
```

### Examples

```
  urlscan pro visibility update <scan-id> -v private
  echo <scan-id> | urlscan pro visibility update - -v public
```

### Options

```
  -h, --help                help for update
  -v, --visibility string   The new visibility of the scan result. public, unlisted, private or deleted.
```
