## urlscan scan dom

Download a dom by UUID

```
urlscan scan dom <uuid> [flags]
```

### Examples

```
  urlscan scan dom <uuid>
  echo "<uuid>" | urlscan scan dom -
```

### Options

```
  -f, --force           Force overwrite an existing file.
  -h, --help            help for dom
  -o, --output string   Output file name (default <uuid>.html)
```
