## urlscan pro live-scan result

Get a result of a live scan

```
urlscan pro live-scan result [flags]
```

### Examples

```
  urlscan pro live-scan result <scan-id> -s <scanner-id>
  echo <scan-id> | urlscan pro live-scan result - -s <scanner-id>
```

### Options

```
  -h, --help                help for result
  -s, --scanner-id string   ID of the scanner (required)
```
