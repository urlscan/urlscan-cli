## urlscan scan open

Open a scan result in your browser by UUID

```
urlscan scan open <uuid> [flags]
```

### Examples

```
  urlscan scan open <uuid>
  echo "<uuid>" | urlscan scan open -
```

### Options

```
  -h, --help   help for open
```
