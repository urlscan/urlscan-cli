## urlscan pro datadump list

Get the list of data dump files

```
urlscan pro datadump list [flags]
```

### Examples

```
  urlscan pro datadump list --time-window days --file-type api --date 20260101
```

### Options

```
  -d, --date string          Date for the data dump (e.g., 20260101)
  -f, --file-type string     File type for the data dump (api, search, screenshot, dom)
  -h, --help                 help for list
  -t, --time-window string   Time window for the data dump (days, hours, minutes)
```
