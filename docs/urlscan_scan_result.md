## urlscan scan result

Get a result by UUID

```
urlscan scan result <uuid> [flags]
```

### Examples

```
  urlscan scan result <uuid>
  echo "<uuid>" | urlscan scan result -
```

### Options

```
  -h, --help   help for result
```
