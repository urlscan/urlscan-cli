## urlscan pro channel get

Get the search results for a specific notification channel

```
urlscan pro channel get [flags]
```

### Examples

```
  urlscan pro channel get <channel-id>
	echo <channel-id> | urlscan pro channel get -
```

### Options

```
  -h, --help   help for get
```
